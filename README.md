# KToolkit

KToolkit 是一个轻量级 Android 开发工具包，旨在提供一系列高性能、易于使用且零第三方依赖的模块，帮助开发者解决日常开发中的痛点。

## 模块列表

### 1. KLog (日志框架)
[![Maven Central](https://img.shields.io/maven-central/v/io.github.flyxinhua/klog.svg?label=Maven%20Central)](https://central.sonatype.com/artifact/io.github.flyxinhua/klog)

KLog 是一个高性能、高度可扩展的日志框架，支持懒加载、组合式输出（Logcat、文件、UDP网络等），有效避免了日志构建的性能浪费。

[→ 了解 KLog 详情](./klog/README.md)

### 2. KBLE (蓝牙框架)
[![Maven Central](https://img.shields.io/maven-central/v/io.github.flyxinhua/kble.svg?label=Maven%20Central)](https://central.sonatype.com/artifact/io.github.flyxinhua/kble)

KBLE 是一个专为 Android 打造的轻量级、高性能的蓝牙低功耗 (BLE) 框架，提供设备中心连接管理、智能扫描、自动重连和生命周期感知等高级特性。

[→ 了解 KBLE 详情](./kble/README.md)
